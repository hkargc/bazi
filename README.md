# Bazi (四柱八字排盘工具)<br>

这是我学习命理时自用的排盘工具.<br>

PC浏览器访问即可,纯JS实现,暂未对移动端进行刻意优化.<br>

当时需要解决以下几个问题,才下定决心手搓了这套系统:<br>

a,批盘记录因为极度涉及隐私,不放心存到第三方服务器.<br>

b,要同时进行多个盘的对比分析,市面上的工具没有此功能.<br>

c,要能支持真太阳时,支持保存批注,支持刑冲合害提示等.<br>

d,便捷的输入界面,符合我的使用习惯.<br>

e,其他.<br>

## 🚀 快速开始<br>
在线演示: https://hkargc.github.io/bazi/ <br>

把项目下载过去放到WEB目录,浏览器访问即可,这个是脱机版.<br>

如果要批注保存功能,则涉及后端数据库的增删改查,需要自己开发了.<br>

后端我用的是PHP实现,没搞过的环境都搭不起来,代码就不放出来了,会搞的看一下JS代码应该就能实现.<br>

我放了一套在公网: https://bz.cipuji.com/ 这是我个人的服务器,同样是无法保证隐私安全的,使用时请自斟酌.<br>

年月日时分秒可以用鼠标滚轮操作,个人觉得挺便捷的.<br>

由于经常要去贴吧或论坛找案例学习,把出生时间复制到"智能解析输入框"就能排盘了.<br>

触发排盘后,浏览器地址栏会被更改,复制链接地址转发即可实现排盘的分享.<br>
